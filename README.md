# Nithin Songala — Portfolio

Personal portfolio site. Built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com). See [Spec.md](./Spec.md) for full requirements.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Before you go live

1. **Photo** — Add your photo at `public/images/nithin.jpg` (About section).
2. **Resume** — Add `public/resume.pdf` for the Download Resume button.
3. **OG image** — Add `public/images/og-preview.png` for social previews.
4. **GitHub Pages** — In repo **Settings → Pages**, set source to **gh-pages** branch. For custom domain, add `nithinsongala.me` in Pages settings and ensure DNS has the correct CNAME/records.

## Deploy

Push to `main`; the GitHub Action builds and deploys to the `gh-pages` branch.

## Commands

| Command       | Action        |
| ------------- | ------------- |
| `npm run dev` | Start dev server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
